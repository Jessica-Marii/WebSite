Computer Build Notes

Mother Board:

Integrated- all parts are connected but still has a few expansion slots (non removable graphics card)
Non-integrated- only expansion slots (removable graphics card)
Server boards- handles storing data, processing speeds
  • Socket platform
  • Chipset
  • LAN controllers
  • Server management
  • Integrated manageability
Form Factor- size of the board ATX(12 x 9.6 in) MicroATX(9.6 x 9.6 in) minATX(6.7 x 6.7)
Chipset- processors CPU (Intel core i9-9000k 3.6 GHz 8-core Processor)
Expansion slots- PCI or express, crossfire, video cards
Ports- USB (3.1 Type C), HDMI (2.0 4K), display port (144hz), Ethernet
Dual-Channel Technology- allows you to multiply throughput twofold (RAM)
RAID Support- Allows data to be stored across multiple hard drives
  • RAID 1 setup consists of at least two drives that are mirrored to contain the exact same information
  • RAID 5 uses a combination of striping and parity that is distributed across drives
  • RAID 6 in a four-drive setup you lose half of your storage space for parity, though your data will still be intact in the event of simultaneously losing two drives
  • RAID 10 (also known as RAID 1+0) is best suited for four drives. Instead of relying on either mirroring or striping, it actually involves both, hence the 1+0. Data is essentially striped between two sets of mirrored disks, creating a system that can handle a single drive failure in either set of drives or a simultaneous drive failure in both sets. What it cannot deal with is a simultaneous drive failure of one set.

CPU:

  • performs calculations, actions, and runs programs. (fetch from RAM, decode, and execute)
  • (dual, quad, or even octo)-core processor for example is really just two separate CPUs on a single chip. By increasing the amount of cores, CPUs are able to handle multiple processes simultaneously. This had the desired effect of increasing performance and reducing processing time.
Prefixes:
  • Mobile processor prefixes have “U” for ultra-low power, “HQ” for high performance graphics, and “HK” for high performance graphics with the ability to overclock.
  • Desktop prefixes include “K” for ability to overclock,and “T” for optimized power.
Bandwidth - Allows so many bits to be set and processed to the CPU at a time (64 or 32 bit's)
Thermal Power Design- is a measure of maximum power in Watts your CPU will consume.
L2/L3 Cache- this will control how fast the CPU can process
Frequency- refers to the operating speed of the processor.
Core Clock - is how fast the computer can run
Boost Clock- is the fastest the computer can run

CPU Cooler:

Air:
  • Metal heatsinks and fans, come in all shapes and sizes and varying thermal dissipation capacities (usually listed as TDP).
  • Can be larger in size with limited cooling
Closed -loop(AIO):
  • Liquid
  • Can be (but aren’t always) quieter than air coolers
  • Can have an easier fit

Space & Power:

Storage
  • When looking for storage you will have to see what will phyically fit the build. Also you need to decide to use a RAID setup. RAID are typically not used for home PC's.
  • For the best performance it is ideal to get the same brand SSD or hard drive if having multiple in the PC.
Memory(RAM) - 32 gb and up is ideal for a gaming computer match pin count to the mother board
Power Supply- If you wish to add more expansion cards in the future, I would recommend getting a higher wattage power supply for any adjustments later in the future.
