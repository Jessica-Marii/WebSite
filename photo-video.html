Editing Software

Window's Movie Maker

I have used this software for most of my videos when I first started. It does not have many features and can take some time to edit something good. This software can trim, cut, change volume, slowdown, and speed up videos. It is good when just starting out or for a more simpler video.

Final Cut Pro

Final Cut Pro software is run on a Apple oprerating system. I would say is a lot better than Windows Movie Maker due to the fact you can adjust the color of the footage and adjust the background sound. Just those two features alone made a big difference on how my video edits came out. It was also a learning experaince to use Apple's software.

DaVinci Resolve

This is my favortie out of all of the editing software's I used. It has the abilty to edit and export in 4K, also the editing layout is laid is nicely done. It also has color and sound adjustment's.

Adobe Premiere

I have learned and used Adobe Premiere software for a elective class towards my Bachlors. I find it similar to DaVinci Resolve, as a advanced editing program it also provides you to make elaborate edits to a video edit.
